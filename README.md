# 🧠 Exercícios Programação

Repositório dedicado à prática contínua de lógica e resolução de problemas, com exercícios de múltiplas linguagens e plataformas. Também contém alguns exercícios de linguagens de marcação como HTML e CSS. Cada commit representa um exercício concluído, documentando minha evolução como desenvolvedor.

##  📜 Sobre o Projeto
Este repositório reúne soluções de desafios de programação focados em desenvolvimento de raciocínio lógico e algorítmico. Os problemas são extraídos de juízes online e plataformas de programação competitiva, organizados por linguagem e nível de dificuldade.
Funciona também como um registro público da minha jornada de estudos em Ciência da Computação.
